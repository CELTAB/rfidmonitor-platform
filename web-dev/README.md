# RFIDWeb Platform
This interface web of RFIDMonitor Platform
* private
* public

### Version
1.0.0

### Requirements
- NodeJS >= v4.2.2
- Npm >= v2.14.7
- Grunt >= v0.4.5

### Installation

```sh
$ git clone [git-repo-url] rfidmonitor-platform
$ sudo npm install -g grunt grunt-cli
$ cd web-dev
$ npm install
$ grunt prod
```

[RFIDMonitor]: <https://github.com/CELTAB/rfidmonitor>
